## Command

### `simulate`   执行前向过程，输入一个mask，给出printed image

####  options：

+ `--mask  [image_file]`      必须要有，暂时只支持glp格式和png格式
<!-- + `--model`   可选项包括simpleilt，levelset，multilevel，curvmulti，nerul-ilt等。default=simpleilt -->

+ `--output_format`     可选项包括printedNom, printedMax, printedMin。default=printedNom
<!-- + `--output_metrics`  是否输出L2，pvb等指标。可选项包括True，false。 default=False -->
+ `--tile_sizeX`   default=2048
+ `--tile_sizeY`   default=2048





### `optimize`   执行优化过程，输入target，给出一个优化过的mask

#### options：

+ `--target [image_file]`   必须要有，暂时只支持glp格式和png格式
+ `--model`   可选项包括simpleilt，levelset，multilevel，curvmulti，neural-ilt等。default=simpleilt
+ `--output_result`   是否输出最终的printed image。可选项包括True，false。

+ `--output_format`     只有当--output_result=True时生效。可选项包括printedNom, printedMax, printedMin。default=printedNom

+ `--output_metrics`  是否输出L2，pvb等指标。可选项包括True，false。 default=False
+ `--tile_sizeX`   default=2048
+ `--tile_sizeY`   default=2048



### `evalute`   执行评估过程，输入mask和target，给出各个metric

#### options：

+ `--mask  [image_file]`      必须要有，暂时只支持glp格式和png格式

+ `--target [image_file]`    必须要有，暂时只支持glp格式和png格式
